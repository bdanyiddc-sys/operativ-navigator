# Operativ Navigator

Kisvonat / Operativ Navigator – utas, sofőr és admin felület egy repo alatt.

## Struktúra

```
operativ-navigator/
├── frontend/     → Netlify (publish: frontend)
└── backend/      → Render (root: backend)
```

## Gyors linkek

- Frontend deploy: [frontend/DEPLOY.md](frontend/DEPLOY.md)
- Backend API: `backend/server.js` (port 3000)

## Éles URL-ek (példa)

| Szolgáltatás | URL |
|--------------|-----|
| Utas | `https://<netlify-domain>/public/` |
| Sofőr | `https://<netlify-domain>/driver/` |
| Admin | `https://<netlify-domain>/admin/` |
| API | `https://operativ-navigator.onrender.com` |
